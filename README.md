📚 Менеджер книг (учёт прочитанного) – Multi‑Language Implementation
Тестовый репозиторий с реализацией консольного менеджера книг на 7 языках программирования.
Приложение помогает вести учёт прочитанных книг: добавлять, редактировать, удалять, искать, получать статистику и сохранять данные.

📖 Что такое Менеджер книг?
Это личная библиотека для учёта книг. Вы можете хранить информацию о книгах: название, автор, год издания, жанр, статус (прочитана или нет), вашу оценку (от 1 до 10). Приложение позволяет:

Добавлять книги с указанием всех полей.

Просматривать все книги или фильтровать по статусу, жанру, автору.

Редактировать информацию о книге.

Удалять книги.

Получать статистику: общее количество книг, количество прочитанных, средняя оценка, распределение по жанрам.

Сохранять и загружать данные в файл (JSON или сериализованный объект) для постоянного хранения.

Поиск книг по названию или автору (частичное совпадение).

Сортировка по году, оценке, названию.

✨ Особенности
Интерактивное меню с выбором действий.

Все поля валидируются (год от 0 до текущего, оценка 1–10).

Автоматическая генерация ID и даты добавления.

Расширенная статистика: количество по жанрам, средний балл.

Данные сохраняются в формате JSON (или эквивалент) – легко читаемо и переносимо.

Цветной вывод (где поддерживается) для наглядности.

Загрузка данных при старте.
